## ⚡ Quick Deploy (5 minutes)

### Step 1: Install Railway CLI
```bash
npm install -g @railway/cli
```

### Step 2: Login
```bash
railway login
```

### Step 3: Deploy
```bash
railway init
railway up
```

**That's it!** Railway handles everything automatically.

## 📋 Detailed Steps

### 1. Install Railway CLI
```bash
npm install -g @railway/cli
```

### 2. Login to Railway
```bash
railway login
```
Opens browser, sign in with GitHub.

### 3. Initialize Project
```bash
railway init
```

Answer prompts:
- Create a new project? → **Yes**
- Link to existing project? → **No** (first time)

### 4. Deploy
```bash
railway up
```

Railway will:
- Detect your Node.js app
- Build automatically
- Deploy immediately
- Give you a URL
